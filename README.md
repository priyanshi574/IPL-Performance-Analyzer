# IPL Player Performance Analyzer (2008-2025)

## Project Overview
Analyzed 278,000+ IPL ball-by-ball records to identify 
top performers and auction strategy insights.

## Tools Used
- **Python** — Data Cleaning & Analysis (Pandas, Matplotlib, Seaborn)
- **SQL** — Data Querying (SQLite)
- **Tableau** — Interactive Dashboard

## Analysis Done
- Top 10 Batsmen by Total Runs & Strike Rate
- Top 10 Bowlers by Wickets & Economy Rate
- Season Wise Runs Trend (2008-2025)
- Best Death Over Bowlers (Over 16-20)

## Key Insights
- Virat Kohli — Most runs (8671) in IPL history
- SP Narine — Best in both wickets & economy rate
- JJ Bumrah — Most reliable death over bowler
- PD Salt — Highest strike rate (177) among active players
